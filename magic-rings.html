Magic Rings (4-Axis Machining Illusion)

Status: Complete

Summary: Planned and executed a multi-axis machining strategy to create two interlocked rings with a seamless “impossible” appearance.

Overview

This project is a constraint puzzle in tool access, fixturing, and sequencing. The process was developed to avoid witness marks that would break the illusion.

The emphasis was on setup discipline and multi-axis planning, where small errors become visible immediately.

Tools and skills

CAD 4-axis machining CAM Fixturing
Expand details strategy • sequencing • improvements
  • Geometry: ring profiles designed for tool access and consistent finish.
  • Setup planning: index angles and workholding to avoid collisions and remount error.
  • CAM strategy: roughing for removal, finishing passes prioritized for surface quality.

A stronger second iteration would include a formal setup sheet (tools, operations, index angles, inspection checkpoints).
