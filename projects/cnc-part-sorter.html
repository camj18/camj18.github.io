CNC Part Sorter (2-Axis Sorting Gantry)

Status: Complete

Summary: Built a benchtop XY gantry that captures an image of a part, classifies it, moves to a target bin, and releases the part using a servo-actuated gate.

Overview

This project was an end-to-end automation build: mechanism, sensing, decision logic, and repeatable actuation. The goal was a system that can execute a full sorting cycle reliably under controlled conditions.

The platform is a stacked XY stage built from belt-driven linear actuators on 80/20 extrusion. A servo-driven tilt plate provides a robust release mechanism with minimal failure modes.

Tools and skills

Mechatronics MATLAB Automation Prototyping
Expand details mechanism • pipeline • improvements

Design / technical approach

  • Motion system: belt-driven linear actuators assembled into an XY stage.
  • Structure: 80/20 frame bracing to maintain squareness and reduce racking.
  • Sorting hardware: 2×2 bin tray designed as an MVP and easy to scale.
  • Release: servo-actuated tilt gate for consistent drop behavior.

Perception and decision pipeline

  • Capture: consistent camera geometry and lighting.
  • Classify: MATLAB workflow using image processing and a trained detector setup.
  • Actuate: move to bin center → tilt gate → return/home.

What I learned / would improve

The largest gains come from environment control (lighting, camera pose) and calibration. A second iteration would add a coordinate calibration routine and structured test logging.
